# Broken Robot

a simple platformer where you play as a broken robot trying to keep itself running by collecting energy cores

## what it is

you move through levels, avoid spikes, and try not to run out of energy. the robot’s system is unstable so things get harder the further you go

## features

* basic movement (left, right, jump)
* energy system
* spikes and obstacles
* difficulty ramps up over time

## controls

* arrow keys to move
* space to jump

## how i made it

i started with just getting a square moving on screen, then added jumping and gravity. after that i worked on collisions so it could actually land on platforms. then i added spikes, energy, and just kept building from there

## challenges

movement took longer than expected to feel decent. also balancing the game was kinda annoying because it either felt too easy or just unfair. had to tweak it a lot

## what i learned

i learned how game loops actually work and how to manage multiple things updating at once. also got better at debugging when stuff randomly breaks

## if i had more time

i’d add more levels, better visuals, maybe sound, and make the robot feel a bit more alive instead of just a block
